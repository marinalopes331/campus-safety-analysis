# Campus Safety Analysis

## Overview
This repository contains a cleaned dataset examining student perceptions of campus safety at night.

## Data Sources
- Student survey (n=1,200 simulated for classroom use)

## Methodology
- Standardized category labels (e.g., “neutral” → “Neutral”)
- Removed incomplete entries
- Removed non-essential notes column
- Cleaned spacing and formatting inconsistencies

## Limitations
- Data is simulated for instructional purposes
- Self-reported perceptions may not reflect actual safety conditions
- Small sample size in this dataset

## How to Use
- Use this dataset to practice data analysis, visualization, and writing
- Compare perception trends and behavioral responses

## Transparency Note
This dataset has been cleaned for clarity. All cleaning steps are documented above to ensure transparency and reproducibility.
